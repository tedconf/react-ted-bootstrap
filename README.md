# TED Bootstrap (React version)

React version of [TED Bootstrap](http://tedlive.ted.com/swatch)

## Installation

```bash
yarn add @tedprivate/react-ted-bootstrap
```

## Components List

* [`Button`](#button)
* [`Container`](#container)
* [`DataTable`](#datatable)


### Button

`ember-ted-button` equivalent

```
import { Button } from 'react-ted-bootstrap';

const MyComponent = () => {
  <Button>
    My Button
  </Button>
}

```
