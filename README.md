# TED Bootstrap

A React-based design system for internal TED projects.

## Installation

```bash
yarn add @tedconf/react-ted-bootstrap
```

## Components List

* [`Button`](#button)
* [`Container`](#container)
* [`DataTable`](#datatable)


### Button

`ember-ted-button` equivalent

```
import { Button } from '@tedconf/react-ted-bootstrap';

const MyComponent = () => {
  <Button>
    My Button
  </Button>
}

```
